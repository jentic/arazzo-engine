# Docling Scripts

This directory contains scripts and documentation for testing and using Docling with the Arazzo Runner.

## Files

- `docling_execute_example.py` - Example of executing Docling operations
- `test_docling_convert.py` - Tests for Docling PDF conversion
- `test_file_io_capabilities.py` - Tests for file I/O with Docling (multipart/form-data)
- `DOCLING_CLI_EXAMPLE.md` - CLI usage examples for Docling
- `TEST_FILE_IO.md` - Guide for testing file I/O capabilities

## Usage

See individual files for usage instructions.
